Do you need help with Journey?

If you need help with Journey, the Incident IQ system can put you in touch with individuals ready to assist you! Please follow these directions to get assistance:

  • Use the Classlink system (https://launchpad.classlink.com/polk?autosamllogin=1) to login to Incident IQ.
  • Consult the following list of areas for assistance:
    • For help understanding Journey's content (forms, documents, worksheets, or evaluation policies/procedures):
      Software / Online Systems > PCPS Journey > Support - Evaluation Processes (Forms/Worksheets/Deadlines)
    • For help with questions about VAM and State of Florida student performance calculations:
      Software / Online Systems > PCPS Journey > Support - VAM and State Performance
    • For technical issues/pages that have errors:
      Software / Online Systems > PCPS Journey > Employee Issues or Supervisor Issues
